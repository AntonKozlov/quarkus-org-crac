== Installation

If you want to use this extension, you need to add the `io.quarkiverse.quarkus-org-crac:quarkus-org-crac` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.quarkus-org-crac</groupId>
    <artifactId>quarkus-org-crac</artifactId>
</dependency>
----
